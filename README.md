# ByteBolt

#### C# High-level library for byte manipulation, and fast encode/decode of datas


## Usage

#### Namespace
```csharp
using ByteBolt; // api
```
```csharp
using ByteBolt.Core; // internal
```

#### Accepted Types
```
// byte
byte;
byte[];

// int | Int32
int;
uint;

// short | Int16
short;
ushort;

// long | Int64
long;
ulong;

// arithmetic
float;
double;

// other
char;
string;
```
